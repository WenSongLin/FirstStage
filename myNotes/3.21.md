#### 1、访问权限修饰符

|                                 | 修饰         | 作用                                 |
| ------------------------------- | ------------ | ------------------------------------ |
| public 公共的(用的最多)         | 修饰类、成员 | 在项目内都是可以访问的(项目内都有效) |
| protected 受保护的              |              |                                      |
| private  私人的(用的最多)       |              |                                      |
| 默认(default)                   | 修饰类、成员 | 在本包内友好                         |
| static  静态                    |              |                                      |
| final 最后                      |              |                                      |
| abstract 摘要，抽象的，纯理论的 |              |                                      |
